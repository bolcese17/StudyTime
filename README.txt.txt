SEARCH COURSE FUNCTION:
	Input a course name and the average of student time breakdown will be returned. For the purpose of this project, the
classes that can be searched are hard coded with two options "CSC 123" and "CSC 190"

ADD CLASS FUNCTION:
	Input course name, teachers last name, and hours spent on class time, study time, assignment time, and recreational
time. You can also delete a course that you had mistakenly added.
